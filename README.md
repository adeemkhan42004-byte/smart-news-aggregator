# Smart News Aggregator Web App

This is a web-based news application that fetches and displays real-time news articles from various sources using a public API.

## Features
- Real-time news updates using API
- Category-based filtering (Sports, Tech, Business)
- Responsive and user-friendly interface
- Search functionality for news topics

## Tech Stack
- HTML
- CSS
- JavaScript
- News API

## Working
1. Fetches data from a news API
2. Displays headlines dynamically
3. Users can filter news based on categories
4. Updates content in real-time

## Future Improvements
- User login system
- Bookmark articles feature
- Dark mode UI
